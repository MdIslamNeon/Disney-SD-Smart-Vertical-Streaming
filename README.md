# Disney-SD-Smart-Vertical-Streaming

Smart Vertical [TBD Sport] Sports Reels 

Project Overview:

Most sports live streams are produced in the horizontal 16:9 aspect ratio, but mobile audiences prefer the vertical 9:16 aspect ratio. Cropping a horizontal feed to fit a vertical 9:16 display often results in the loss of essential action, such as the player or the ball. Consequently, viewers using phones may miss critical moments, reducing the viewing experience.
 
For this reason, we propose using real-time computer vision and player tracking. We aim to automatically reframe horizontal basketball live streams into vertical formats, ensuring the featured player and ball remain in view, resulting in smoother mobile viewing and more engaging fan experiences. In addition to reframing, the system integrates an automated summarization module that identifies highlights (e.g., dunks, home runs, three-pointers, turnovers) and extracts them into concise clips. These summaries are exported directly in vertical format, making them ready for offline sharing or review

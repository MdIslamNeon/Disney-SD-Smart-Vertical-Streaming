# SETUP INSTRUCTIONS
# ------------------
# 1. Clone the repository:
#       git clone <repo-url>
#       cd Disney-SD-Smart-Vertical-Streaming
#
# 2. Create a virtual environment:
#       python -m venv venv
#
# 3. Activate it:
#       Windows:  venv\Scripts\activate
#       Mac/Linux: source venv/bin/activate
#
# 4. Install dependencies:
#       pip install -r requirements.txt
#
# 5. Install ByteTrack (not on PyPI — install from GitHub):
#       pip install git+https://github.com/ifzhang/ByteTrack.git
#
#    Note: ByteTrack provides the `yolox` module used for
#    `from yolox.tracker.byte_tracker import BYTETracker`.
#    If the above fails, try cloning and installing manually:
#       git clone https://github.com/ifzhang/ByteTrack.git
#       cd ByteTrack && pip install -e . && cd ..
#
# 6. YOLOv8 weights (yolov8n.pt) are downloaded automatically
#    on first run by the ultralytics package.
# ============================================================

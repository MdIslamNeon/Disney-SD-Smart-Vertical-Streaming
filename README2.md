How to run program 
1. Have any ipynb software
2. Run All and wait a bit for ultralytics, its dependencies and files
3. Drag and drop video files (avi, mp4, mov) into content->ultralytics->ultralytics->videos
4. Run last cell to use yolo player detection on all files (~1-2 seconds per frame)
5. View detected files in content->ultralytics->ultralytics->all_videos_output

Preferred software
Make sure you have something that runs ipynb file like colab or jupyter notebook
Have the latest python version 

Have Pip installed
